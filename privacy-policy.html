QuickTrans Privacy Policy

Last Updated: January 2025

Data Collection

QuickTrans does NOT collect, store, or transmit any personal information or user data to external servers. All data processing occurs entirely on your local device.

Local Storage

The extension uses Chrome's local storage API to store the following data locally on your device:

  • Language preference (Chinese or English)
  • Common query parameter configurations (if you choose to save them)
  • Floating window states (position, minimized state) for synchronization across browser tabs

All stored data remains on your device and is never transmitted to any external server or third party.

Data Processing

All functionality is performed locally:

  • Timestamp conversion: All calculations are performed in your browser
  • QR code generation: Generated entirely on your device
  • QR code recognition: Image processing occurs locally, no images are uploaded

No Tracking

QuickTrans does NOT:

  • Track your browsing behavior
  • Collect analytics data
  • Use cookies or tracking technologies
  • Share data with third parties
  • Access your personal information beyond what is necessary for the extension's core functionality

Permissions

The extension requires the following permissions for core functionality:

  • "storage": To save your preferences locally
  • "activeTab": To access the current tab when you interact with the extension
  • "contextMenus": To provide right-click menu options

All permissions are used solely for the extension's stated functionality and no data is collected or transmitted.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the extension's update notes.

Contact

If you have any questions about this Privacy Policy, please contact us through the Chrome Web Store listing.
